# Convertidor de Binario a decimal
| Entrada | Salida |
|---------|--------|
| 11      | 3      |
| 111     | 7      |
| 101     | 5      |

# Comandos
* `npm init -y` para crear un nuevo proyecto de node. Nos creará un package.json
* `npm install jison`  para instalar jison. Generador de analizadores lexicos y sintácticos.
